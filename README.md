# FileBox 产品介绍文档

## 0. 产品下载与安装
**下载软件**：[点击链接](https://github.com/walnut-a/FileBox-Public/releases/tag/Release)下载最新版本，仅支持 macOS。

**安装方法**：双击打开 DMG 文件后，将 FIleBox 图标拖拽到右侧 Application 文件夹。

**使用技巧**：FileBox 完全在本地运行，核心功能是提供快捷访问文件夹的界面，并不会干涉文件的存储与处理，所以并不会有导致文件丢失或损毁的风险。FileBox可以和iCloud完美配合，锦上添花，比如将常用素材放在收藏文件夹，并开启 iCloud 同步，便可以在多个设备上借助 FileBox 快速调用文件。

FileBox 全程通过 [Antigravity](https://antigravity.google/) 构建，下方介绍文档亦由 Gemini 3 Pro生成。

## 1. 产品概述
**FileBox** 是一款专为 macOS 设计的现代化文件管理与中转工具。它旨在解决用户在日常工作中频繁查找、移动和临时存储文件的痛点。通过提供一个始终触手可及的“快捷面板” (Quick Panel) 和精心设计的“中转站” (Transit) 概念，FileBox 让文件的临时存放与流转变得前所未有的流畅与优雅。

FileBox 不仅仅是一个文件浏览器，它是你桌面工作流的高效润滑剂，完美融合了 macOS 原生设计美学与极客级的效率工具特性。

## 2. 核心亮点

### 🚀 快捷面板 (Quick Panel)
FileBox 提供了一个类似 Spotlight 的全局 HUD 快捷面板。
- **随叫随到**：通过全局快捷键（默认 `Ctrl+Option+Space`，可自定义）一键唤出。
- **智能跟随**：面板出现位置可配置为“主屏幕”、“当前活动窗口所在屏幕”或“鼠标位置”，确保文件总是在你最顺手的地方。
- **迷你文件流**：在不打断当前工作流的情况下，快速浏览、拖拽或打开最近的文件。

### 🚉 文件中转站 (Transit)
不再让桌面堆满临时文件。FileBox 的“中转站”是你处理临时文件的最佳场所。
- **临时存放**：将需要稍后处理、发送或移动的文件暂时拖入中转站。
- **保持整洁**：不仅清理了桌面，也让文件归档更有条理。

### ⭐ 智能收藏夹 (Favorites)
不仅仅是简单的快捷方式。
- **拖拽行为**：支持配置拖拽文件到收藏夹时的默认行为（移动或复制），让常用素材的收集与复用更加高效。

## 3. 详细功能清单

### 📂 文件管理
- **多位置支持**：
    - **内置区域**：下载 (Downloads)、中转站 (Transit)、收藏夹 (Favorites)。
    - **自定义目录**：支持添加多达 10 个自定义文件夹到侧边栏，快速访问项目路径。
- **视图模式**：提供 **列表 (List)** 和 **网格 (Grid)** 两种视图，满足不同文件的浏览需求。
- **智能排序**：支持按修改时间、名称、大小、种类等多种方式排序。
- **文件过滤**：内置高效过滤器，一键筛选图片、音频、视频或文档，快速定位目标文件。

### ⚡️ 高效交互
- **全局拖拽**：完美支持 macOS 的拖拽操作。从 Finder 拖入文件进行管理，或从 FileBox 拖出文件到其他应用。
- **快捷键支持**：
    - 快速切换过滤器 (默认 `Cmd+P`)。
    - 自定义唤起面板的快捷键。
- **原生体验**：采用 SwiftUI 构建，拥有丝滑的动画效果和 Material Vibrancy 设计，完美融入 macOS 生态。

### ⚙️ 个性化设置
- **自定义路径**：不仅可以自定义侧边栏文件夹，甚至可以为“中转站”和“收藏夹”指定物理磁盘上的任意路径。
- **启动管理**：支持开机自启，确保工具时刻待命。
- **多语言支持**：内置完善的中英文国际化支持。
- **菜单栏常驻**：通过菜单栏图标也可快速访问或唤起面板。

## 4. 使用场景

- **设计师整理素材**：将网上下载的图片素材先拖入“中转站”，分类筛选后，批量拖入项目文件夹或“收藏夹”。
- **开发人员管理文档**：将常用的 API 文档或配置文件路径固定在侧边栏，通过快捷面板随时查阅。
- **办公清理桌面**：每天下班前，将桌面的临时截图和文档一键拖入 FileBox 进行归档，通过快捷键唤出面板快速清理下载文件夹。

## 5. 系统要求
- **操作系统**：macOS 13.0+
- **架构**：支持 Apple Silicon (M1/M2/M3) 及 Intel 芯片
